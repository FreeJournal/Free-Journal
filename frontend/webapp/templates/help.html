Help

FreeJournal is a decentralized system for publishing documents. A document can be any type of file. These files are organized in "collections," which group the files under a common title.

Using the FreeJournal web interface is super easy. To see available documents in a collection, click the link for that collection.

You can then click a document title to download that document.

Sometimes it takes a while for a collection or document to appear on the FreeJournal web interface once it has been submitted. Large documents can take an especially long time.
